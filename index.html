Privacy Policy

Dungeons & Dragons

Last updated: January 2026

This Privacy Policy explains how the game Dungeons & Dragons ("the Game", "we", "our", or "us") collects, uses, and protects information.

Information We Collect

The Game does not collect personal information such as real names, email addresses, phone numbers, precise location, or contact details.

The Game collects only game-related and functional data, including:

  • In-game currency (such as coins)
  • Heroes, items, and gameplay progress
  • Login identifiers required to access the Game

This data is used solely to provide core game functionality.

Login Data & Cloud Database

The Game uses a cloud database hosted on Microsoft Azure.

The database stores:

  • Game login identifiers
  • Player progress and gameplay data

Login data is used only to authenticate players and associate gameplay progress with their game account. No real-world identifying information is collected.

Multiplayer & Online Services

The Game includes online multiplayer functionality.

Multiplayer services are hosted on Amazon Web Services (AWS) and use the Mirror networking framework.

Only gameplay-related data is transmitted between players and servers. No personal data is collected during multiplayer sessions.

Server Region Selection

To provide the best multiplayer experience, the Game may determine the player’s approximate region, such as continent or country.

This information is used only to:

  • Select the nearest or most suitable multiplayer server
  • Reduce latency and improve connection quality

We do not collect or store precise location data such as GPS coordinates, exact addresses, or real-time location.

In-App Purchases

The Game uses Google Play In-App Billing for optional purchases.

All payments are processed securely by Google. We do not collect or store payment or financial information.

Children’s Privacy

The Game is intended for a general audience, including children under 13.

We do not knowingly collect personal information from children under 13. Only non-personal data required for gameplay, login, and multiplayer functionality is stored.

If you believe personal data has been collected unintentionally, please contact us and it will be deleted.

Data Sharing

We do not sell, rent, or share user data with third parties.

Data is shared only with service providers necessary to operate the Game, such as cloud hosting and database services.

Data Security

Reasonable technical and organizational measures are used to protect game data.

However, no method of electronic storage or transmission is completely secure.

Data Retention

Game-related data is retained only as long as necessary to provide the Game’s functionality or until the user stops using the Game.

Changes to This Policy

This Privacy Policy may be updated from time to time. Any changes will be posted on this page.
